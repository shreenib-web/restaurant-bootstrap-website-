# Foodie Restaurant Website

## Description
Foodie is a responsive restaurant website built with HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript. It is designed for a frontend developer portfolio, featuring a modern layout, clean typography, and interactive restaurant pages.

## Features
- Responsive multi-page website
- Home, Menu, About, Reservation, and Contact pages
- Featured dishes and menu cards
- Customer testimonials carousel
- Reservation form with validation and success alert
- Contact form validation
- Smooth scrolling and mobile menu behavior
- Dynamic current year in footer
- Modern responsive design with Bootstrap utilities
- Polished hero sections and animated section reveals
- Enhanced hover effects, card transitions, and modern UI details
- Newly redesigned contact page with a richer layout and visit planning section

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript
- Bootstrap Icons
- Google Fonts

## How to Run Project
1. Clone or download the project folder.
2. Open the `restaurant-bootstrap-website` folder in VS Code.
3. Open `index.html` in a browser or use Live Server.
